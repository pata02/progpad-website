Kod du kan ta på.
Lekfullt. Logiskt. Lärorikt.

Ett fysiskt verktyg som öppnar dörren till programmering för alla, oavsett ålder eller bakgrund. ProgPad förvandlar kod till något du kan se, röra och förstå.

Vad är ProgPad?

lightbulb
ProgPad började som ett examensarbete på KTH med syftet att utforska hur programmeringsundervisning i grundskolan kunde göras mer konkret och begripligt.
control_camera
Målet var att skapa ett verktyg som kombinerar teknik och pedagogik på ett sätt som både engagerar och underlättar inlärning – särskilt för yngre elever och nybörjare.
check
ProgPad är resultatet: en fysisk enhet där elever sätter samman program med hjälp av kodklossar – varje kloss representerar ett programmeringsbegrepp som till exempel en variabel, loop eller villkorssats. Detta ger ett konkret och intuitivt sätt att närma sig programmering, där logik möter lek och abstrakta begrepp blir greppbara.
neurology

Brett undervisningsstöd

Ger stöd för undervisning i allt från grundläggande till avancerade programmeringskoncept – som variabler, loopar, villkor, logiska operatorer och booleska uttryck. Allt i ett enkelt men uttrycksfullt programmeringsspråk.

model_training

Omedelbar feedback

Direkt feedback via ljud, ljus och display visualiserar programmets körning i realtid. Elever följer varje steg, ser hur variabler förändras och förstår programmets logik utan att tappa tråden.

cloud_off

Klassrumsvänlig design

Kräver varken dator eller extern mjukvara som ofta strular – allt sker direkt på enheten. Den är snabb att komma igång med, enkel att använda och fri från distraktioner, vilket låter eleverna fokusera helt på programmeringen.

Utforska ProgPad

Använd markörerna för att visa mer information om ProgPads funktioner.

Rutnät för klossar
Här placeras programmeringsklossar rad för rad för att bygga ett program.
Radindikator
Bredvid varje rad finns en lysdiod som bland annat visar vilken rad som körs under programmets gång.
Start / Reset-knapp
Används för att starta, pausa och återställa programmet som användaren har byggt.
Hastighetsreglage
Justerar hur snabbt varje rad i programmet körs.
Display
Visar bland annat variabelvärden under programmets gång och information om syntaxfel med mera.
Input-switchar
Kan användas som input till programmet i form av ett booleskt värde (sant eller falskt), för att exempelvis skapa uttryck i villkorssatser.
Output-lysdiod
Kan användas som output till programmet som användaren skapar och tändas i olika färger med hjälp av lys-klossar.
Högtalare
Kan styras genom programmet som användaren skapar med hjälp av ljud-klossar.
